# notec

notec is a nano/vim clone for windows
im too lazy to port it to linux

## Build

```bash
make
```

## Usage

```bash
notec [file name]
```
Press ESC to change between writing and CTRL mode.

While in CTRL mode:

q -> to quit without saving

w -> save

f -> find

c -> copy

c -> paste
